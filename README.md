## ES7（ECMAScript 2016）
主要是增加了两个新特性：Array.prototype.include和取幂运算符。   

## ES8（ECMAScript 2017）
主要变化包括：Object.values/Object.entries、字符串填充、Object.getOwnPropertyDescriptor、尾随逗号、异步函数、共享内存和原子等。
